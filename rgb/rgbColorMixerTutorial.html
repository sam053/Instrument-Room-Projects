RGB Color Mixer Tutorial

This project is a simple RGB LED color mixer intended for beginners in electronics. This project will help you understand the foundation of analog color mixing, and can be applied to many other LED applications.It can be contructed using basic components found in most electronics kits or bought seperatly inexpensively.

Parts List:

  • (x1) 330 &Omega Resistor
  • (x1) 560 &Omega Resistor
  • (x3) 10 k&Omega Potentiometers
  • (x1) RGB Anode LED
  • Breadboard and Wires
