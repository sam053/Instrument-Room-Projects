Arduino / Xbee Wireless Communication

Project Created by: Sam Butchko and Gavin Marshall

A tutorial aimed at explaining the setup and basic method for transmitting data wirelessly between Xbee SR2 chips, using an Arduino intermediary device. This tutorial makes use of one Router and one Coordinator that receives the stack from of the current operations on the Router Xbee.

Parts List

  • (1x) Arduino Mega
  • (1x) Arduino Uno
  • (2x) Xbee Transceiver
  • Breadboard and Passive Components

Documents

  • Arduino Xbee Setup Tutorial
  • Xbee Stack Frame Data Table
  • Coordinator Arduino Code
  • Router Arduino Code
  • LTSpice Model
